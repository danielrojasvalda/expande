iZi Facturación te permite realizar facturas con el formato legal boliviano de manera automática con cada orden.

== Description ==

iZi Facturación es un plugin diseñado para funcionar de la mano de tu suscripción de iZi, en app.izifacturacion.com

Realiza facturas de manera automática con cada orden realizada, ya sea a través de una pasarela de pago o a través del gestor de órdenes.

Este plugin es necesario si deseas instalar nuestra pasarela de pagos.

== Installation ==


1. Instala este plugin desde el administrador de plugins de tu sitio web, búscalo como "iZi Facturación". Como método alternativo, puedes descargarlo, y subirlo por FTP a la carpeta `/wp-content/plugins/izi-wordpress`.

2. Activa el plugin desde la pantalla de plugins de WordPress.

3. Una vez instalado el plugin, puedes dirigirte a \"Ajustes\" -> \"Configuración iZi\".
Aquí podrás llenar los datos de tu cuenta, para obtener las llaves puedes entrar a https://app.izifacturacion.com/#/integraciones/lista , seleccionar \"Wordpress\", y podrás ver las llaves que necesitas.
